優先隊列

1. 存儲

優先隊列邏輯上是平衡二叉堆，近似於二叉樹。實際存儲是數組。

父子節點關係

  1. 若父節點在數組中位置爲 nnn，則左孩子位置爲 2n+12n + 12n+1， 右孩子位置 2n+22n + 22n+2
  2. 同理，若孩子節點位置爲 nnn，則父節點爲 n−12\frac{n-1}{2}2n−1​ 或者 (n-1)>>>1

節點間關係

  1. 父節點小於等於所有子節點
  2. 同一級的左右節點不需要有序，而是在輸出的時候判斷

判斷葉子節點與非葉子節點

  • 葉子節點: index >= size >>> 1 下標大於等於一半即是葉子節點
